# Secret Santa server

L'attribution des noms se fait en 3 phases :
- Phase 0 où le serveur est actif mais ne fait rien
- Phase 1 où le serveur inscrit les gens qui se connectent
- Phase 2 où le serveur envoie aux gens le nom de la personne à qui ils offrent un cadeau

Le pilotage des différentes phases se fait via le fichier admin.py
